## ❓ FAQ

**Q: Where do I start?**  
A: Read [SETUP.md](SETUP.md) → Run `npm run dev`

**Q: How do I upload a resume?**  
A: Go to http://localhost:3000/analyze → Use UploadBox

**Q: Why is the API not connecting?**  
A: Backend might be down. App uses mock data as fallback.

**Q: Can I deploy to Vercel?**  
A: Yes! See [DEPLOYMENT.md](DEPLOYMENT.md#vercel-recommended)
